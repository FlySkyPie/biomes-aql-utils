# Biomes AQL (Asset Query Language) utilities

This module is extracted from [ill-inc/biomes-game](https://github.com/ill-inc/biomes-game).

> [!NOTE]
> The original document from Biomes didn't explain specific function for the module, "AQL (Asset Query Language) utilities" just temporary name by guess.

## Install

```shell
npm install @flyskypie/biomes-aql-utils
```

## Build

### Generate Typescript from Python

```shell
pdm run gen
```

### Build CLI tool

```shell
pdm run build
```
